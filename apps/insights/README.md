# @sker/insights - 数据分析工具

分析画布使用数据，提供思维模式洞察和效率优化建议。

## 功能特性

- 📈 使用行为分析
- 🧠 思维模式识别
- ⚡ 效率指标监控
- 🎯 个性化建议
- 📊 团队协作分析

## 技术栈

- React 18 + TypeScript
- ECharts 数据可视化
- Python 数据分析后端
- ClickHouse 时序数据库
- TensorFlow.js 模式识别


## 开发
- 包管理工具使用 pnpm
```bash
pnpm run dev
pnpm run build
pnpm run monitor
```
