# @sker/publisher - 导出服务应用

将画布内容导出为各种格式的专业工具，支持批量处理和模板定制。

## 功能特性

- 📄 多格式导出 (PDF/PNG/SVG/MD)
- 🎨 自定义导出模板
- 📦 批量导出处理
- 🔗 分享链接生成
- 📊 导出统计分析

## 技术栈

- Node.js + Express
- Puppeteer PDF生成
- Canvas API 图像处理
- Sharp 图片优化
- AWS S3 文件存储


## 开发
- 包管理工具使用 pnpm

pnpm run dev
pnpm run build
